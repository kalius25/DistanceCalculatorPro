Done Definition

Một module chỉ được đánh dấu FREEZE khi đáp ứng:

Biên dịch/chạy được.
Đúng kiến trúc phân tầng.
Đúng trách nhiệm (SRP).
Không vi phạm Coding Standard.
Không vi phạm Design Decision.
Có type hints cho API công khai.
Có docstring cho API công khai.
Có kiểm thử phù hợp (khi module cần kiểm thử).
Không còn lỗi mức Critical.

Các khuyến nghị mức Recommended hoặc Optional có thể được giữ lại nếu việc sửa không mang lại lợi ích tương xứng.


Definition of Done (DoD)

Một module chỉ được Freeze khi đạt đủ:

Code Review đạt.
Kiến trúc đúng.
Type Hint đầy đủ.
Docstring đầy đủ.
Unit Test pass.
Không có TODO/FIXME.
Không Hardcode.
Không Magic Number.
Không Circular Dependency.
