PB-002 – Engine Exception Mapping
Mục tiêu

GoogleMapsEngine không được để lộ exception của Playwright hoặc Python ra ngoài API của Engine.

Sau PB này, contract của Engine sẽ là:

GoogleMapsEngine.find_routes()

    Success
        ↓
    list[RouteOption]

    Failure
        ↓
    DistanceCalculatorError (hoặc subclass)

Không còn:

TimeoutError
playwright.sync_api.Error
RuntimeError
ValueError

lọt ra ngoài.

Công việc 1 — Xác định boundary

Trong GoogleMapsEngine.find_routes() chỉ nên có hai loại exception đi ra:

ParserException
EngineException

Giải thích:

ParserException
Parser đã chuẩn hóa lỗi của riêng nó.
Engine không nên "bọc" lại.
EngineException
Mọi lỗi phát sinh trong quá trình điều khiển trình duyệt.
Bao gồm timeout, navigation, click, selector...
Công việc 2 — Exception Mapping

Tôi đề xuất quy tắc sau.

Timeout
except TimeoutError as ex:
    raise EngineException(
        "Google Maps request timed out.",
        error_code=ErrorCode.ENGINE_TIMEOUT,
        cause=ex,
        context={
            "timeout": request.timeout,
        },
    ) from ex
Playwright Error
except PlaywrightError as ex:
    raise EngineException(
        "Google Maps browser operation failed.",
        error_code=ErrorCode.ENGINE_ERROR,
        cause=ex,
    ) from ex
ParserException
except ParserException:
    raise

Không bọc lại.

Unexpected Exception
except Exception as ex:
    raise EngineException(
        "Unexpected engine error.",
        error_code=ErrorCode.ENGINE_ERROR,
        cause=ex,
    ) from ex

Điểm khác với Provider là:

Provider không bắt Exception.
Engine là boundary giữa thư viện bên ngoài (Playwright) và ứng dụng, nên việc chuẩn hóa các exception không mong đợi thành EngineException là hợp lý.
Công việc 3 — Context

Ngay từ PB-002, tôi đề xuất chuẩn hóa context tối thiểu:

context={
    "origin": request.origin,
    "destination": request.destination,
    "travel_mode": request.travel_mode.value,
    "timeout": request.timeout,
}

Những thông tin này đủ để debug mà không đưa dữ liệu dư thừa vào exception.

Tiêu chí hoàn thành PB-002

PB-002 được coi là hoàn thành khi đáp ứng tất cả các điều kiện sau:

GoogleMapsEngine.find_routes() chỉ ném ParserException hoặc EngineException.
Tất cả lỗi từ Playwright/Python được chuyển thành EngineException.
EngineException luôn chứa error_code, cause và context phù hợp.
Không thay đổi public API của GoogleMapsEngine.
Toàn bộ test hiện có vẫn pass trước khi bổ sung test mới ở PB-004.
